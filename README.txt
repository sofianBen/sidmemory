Procédure d'installation :
  - dézipper le contenu du dossier 'src' dans /var/www/l3sid/


Description des différents répertoires et fichiers

Le dossier SQL contient tout le code développer durant le projet pour la partie SQL avec : 
  - creation.sql	
  - grant.sql
  - insertion.sq
  - procedures.sql
  - triggers.sql
  - vues.sql
  
Le dossier src contient tout le code qui doit se trouver sur le serveur web. Il est composé des fichier suivants :
  - collections (contient toutes les collections d'image utilisées)
  - db (contient le fichier de connection à la BD)
  - des fichiers images:
    - PaireTrouvee.png
    - face_cache.jpg
    - memory.jpg
    
  - les fichiers php:
    - ajoutcoup.php
    - classemaine.php
    - classement.php
    - classglobal.php
    - classjour.php
    - connexion.php
    - deconnexion.php
    - grillemono.php
    - grillemulti.php
    - historevoir.php
    - historevoirbis.php
    - historique.php
    - index.php
    - inscription.php
    - jouer.php
    - multi.php
    - multigame.php
    - regles.php
    - sologame.php
    - termiparti.php
    
  - le fichier css: index.css
  
  - les fichiers Javascript:
    - srcmulti.js
    - srcsolo.js
